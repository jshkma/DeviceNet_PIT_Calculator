사용 방법 및 주의사항

입력 정보 확인 위치 (SyConDN)

항목 SyConDN 위치
Baud RateMaster → Bus Parameters → Baud Rate
EPR 값Slave → Connection → EPR
MAC IDSlave → Device Configuration
Input SizeSlave → I/O → Input Size (bytes)
Output SizeSlave → I/O → Output Size (bytes)
PIT 입력Master → Bus Parameters → Poll Interval Time

주요 판단 기준

상태 의미
녹색 판정PIT 값 안전. 그대로 적용 가능
노란색 판정EPR 상한 근접. EPR 상향 또는 노드 감소 권장
빨간색 판정설정 불가. 통신속도 업그레이드 필요
노란색 행 (표)8 bytes 초과 노드. I/O 크기 최적화 권장

* 계산 결과는 이론값입니다. 실제 환경에서는 케이블 품질, 노이즈 등에 따라 차이가 있을 수 있습니다.
* 설정 변경 후 최소 24시간 이상 모니터링을 권장합니다.
